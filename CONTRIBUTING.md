# Contributing

## Branch names

- `feature/local-saving`
- `fix/fire-microphone`
- `design/intro-animation`

## Commit messages

Use short action-based messages:

- `Add local story saving`
- `Fix sticker drag bounds`
- `Refine microphone glow`

## Standards

- Keep comments around major systems.
- Preserve large touch targets.
- Keep child-facing language simple.
- Avoid unnecessary dependencies.
- Do not add unlicensed third-party artwork.
- Test desktop and tablet layouts.
